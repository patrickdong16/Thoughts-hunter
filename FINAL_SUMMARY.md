## 🚀 完整启动流程

### 第一步：后端（必须）

```bash
# 1. 安装PostgreSQL
brew install postgresql@15
brew services start postgresql@15
createdb thoughts_radar

# 2. 初始化数据库
cd backend
npm run init-db

# 3. 启动后端
npm run dev
```

### 第二步：CMS（内容管理）

```bash
# 新终端
cd cms
npm run dev
```

访问：http://localhost:5173

### 第三步：移动端（用户端）⭐

```bash
# 新终端
cd mobile
npx expo start
```

然后：
- 按 `i` 打开iOS模拟器
- 按 `a` 打开Android模拟器
- 扫码在真机上运行
