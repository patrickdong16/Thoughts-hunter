# 思想雷达内容规范

> 详细规范请参照：`.agent/rules/thoughts_radar_rules.md`

## 快速参考

### 来源筛选（三选一或并存）

| 类型 | 示例 |
|------|------|
| 权威筛选 | 诺贝尔奖、NYT深度、Foreign Affairs |
| 顶级头脑 | 书籍、TED演讲、播客访谈 |
| 思想材料 | 结构解释、理论框架、价值冲突 |

### 内容结构

```
标题: [观点鲜明的总结]
作者: [姓名]
简介: [身份 + 主要作品/思想]
出处: [时间 · 来源 · 场合]

正文:
- 第1段: 核心观点
- 第2-3段: 主要论据（具体案例/数据）
- 末段: 社会影响/结论

关键词: [3-5个]
```

### 字数要求
- 最低：500字
- 推荐：800-1500字

### UI颜色速查
- 主色：`#00ff88`
- A极：`#4a9eff`
- B极：`#f0a500`
