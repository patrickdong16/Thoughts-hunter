# 🚀 快速启动 - 思想雷达 CMS

## ⚡ 一分钟启动

```bash
# 1. 确保后端已运行
cd backend
npm run dev

# 2. 新终端 - 启动CMS
cd cms
npm run dev
```

访问：
- **CMS**: http://localhost:5173
- **后端API**: http://localhost:3000

---

## 📋 完整启动流程

### 步骤1：启动后端API ✅

如果后端还没运行：

```bash
# 进入后端目录
cd backend

# 确保数据库已初始化（首次运行必须）
npm run init-db

# 启动后端
npm run dev
```

应该看到：
```
╔═══════════════════════════════════════╗
║      思想雷达 API Server              ║
╚═══════════════════════════════════════╝

Server running on: http://localhost:3000
```

### 步骤2：启动CMS ✨

打开**新终端窗口**：

```bash
# 进入CMS目录
cd cms

# 启动开发服务器（依赖已在创建时安装）
npm run dev
```

应该看到：
```
VITE v7.x.x  ready in xxx ms

➜  Local:   http://localhost:5173/
```

### 步骤3：访问CMS 🎉

在浏览器打开：**http://localhost:5173**

## 🎯 首次使用指南

### 1. 查看现有内容

- 打开CMS，默认显示"内容管理"页面
- 应该能看到seed数据中的6条内容（今天的日期）
- 按日期分组显示

### 2. 添加第一条内容

点击"✨ 添加内容"按钮：

```
发布日期: 2026-01-20（明天）
频段: T2
立场: A
标题: 开源AI模型正在改变游戏规则
作者姓名: Andrew Ng
作者头像: AN
作者简介: AI教育家、Coursera联合创始人
出处: DeepLearning.AI 博客
正文: （输入至少500字的内容）
关键词: 开源AI, Llama, 民主化
```

点击"创建" → 成功！

### 3. 编辑内容

- 在列表中找到刚创建的内容
- 点击"编辑"按钮
- 修改任何字段
- 点击"更新"保存

### 4. 管理频段TTI

- 点击顶部"📊 频段管理"
- 找到任意频段，如"T1"
- 点击"编辑"
- 修改TTI值（比如从85改为90）
- 点击"保存"

## ⚠️ 常见问题

### Q: CMS显示"加载失败"？

**原因**：后端可能没有运行或端口不对

**解决**：
```bash
# 检查后端是否运行
curl http://localhost:3000/health

# 如果报错，启动后端
cd backend && npm run dev
```

### Q: 创建内容时提示"至少500字"？

**原因**：正文内容不足500字

**解决**：
- 看表单中的字数计数："325/500字"
- 还需要 175 字会显示在下方
- 继续添加内容直到≥500字

### Q: 提示"Item already exists for this date and frequency"？

**原因**：同一天同一个频段只能有一条内容

**解决**：
- 修改发布日期，或
- 选择不同的频段

### Q: 端口5173被占用？

**解决**：Vite会自动尝试下一个端口（5174, 5175...）

## 📱 使用提示

### 快捷操作

1. **快速添加**：点击右下角绿色圆形 `+` 按钮
2. **快速编辑**：直接在卡片上点击"编辑"
3. **查看统计**：添加表单顶部会显示今日已有多少条

### 内容规范

1. **每日6-8条**：系统会在达标时显示 ✓
2. **正文500字+**：有实时字数统计
3. **关键词**：用逗号分隔，如"AI, GPT, 创意"

### 效率技巧

1. **复制粘贴**：可以从Word/文档复制长文本
2. **频段选择**：会自动显示对应的核心问题
3. **立场选择**：会显示A极和B极的具体描述
