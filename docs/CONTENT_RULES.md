# 思想雷达内容规范 (Content Rules v4.0)

本文件定义内容生成与发布的核心规范。**每次生成内容前必须参照此文件。**

---

## � 发布日类型 (Day Types)

思想雷达将每日发布分为两种类型：**普通日** 和 **主题日**。

| 类型 | 标识 | 说明 |
|------|------|------|
| **普通日 (Regular Day)** | 🔵 | 常规运营模式，均衡覆盖六大频段 |
| **主题日 (Theme Day)** | 🔴 | 针对重大国际活动的专题聚焦模式 |

---

## 🔵 普通日规则 (Regular Day Rules)

适用于：日常运营，无重大事件时期

### 内容配额

| 项目 | 要求 |
|------|------|
| **总数量** | 6-8 条/天 |
| **频段分布** | T/P/H/Φ/F/R 各1条 (1:1:1:1:1:1) |
| **视频内容** | 1-2 条/天 |
| **字符要求** | 生成≥700字 / 验证≥300字 |

### 来源多样性
- 必须覆盖不同作者、不同媒体
- 避免单一来源主导

---

## 🔴 主题日规则 (Theme Day Rules)

适用于：重大国际论坛、峰会、大会、历史性事件期间

### 触发条件

以下情况可启用主题日模式：

| 类别 | 示例事件 |
|------|---------|
| **国际论坛/峰会** | 达沃斯世界经济论坛、APEC峰会、G20峰会、联合国大会 |
| **科技大会** | CES、MWC、Google I/O、Apple WWDC、OpenAI DevDay |
| **学术/政策会议** | 诺贝尔周、慕尼黑安全会议、博鳌亚洲论坛 |
| **重大历史事件** | 领导人会晤、重大政策发布、历史性协议签署 |
| **行业里程碑** | AGI突破性进展、重大产品发布、市场剧变 |

### 内容配额（放宽限制）

| 项目 | 主题日规则 | 与普通日对比 |
|------|-----------|-------------|
| **总数量** | 8-12 条/天 | ↑ 增加 50% |
| **频段分布** | **不限** | 🔓 解除 1:1:1:1:1:1 限制 |
| **视频内容** | **不限** | 🔓 可全部为视频 |
| **来源类型** | 聚焦主题 | 允许密集同源 |

### 🌐 拓展来源规则 (Expanded Sources)

> 🔑 **核心原则**: 主题日**不限于预设人物名单**，任何在活动中发表高质量/有影响力观点的领袖都可入选。

| 类别 | 可入选人物示例 |
|------|--------------|
| **国家领导人** | 各国元首、政府首脑、部长级官员 |
| **央行/财金** | 央行行长、财政部长、IMF/世行负责人 |
| **国际组织** | 联合国官员、WTO总干事、WHO负责人 |
| **商业领袖** | 跨国公司CEO、行业巨头创始人 |
| **学术/智库** | 顶尖学者、智库负责人、诺奖得主 |
| **科技大佬** | AI实验室负责人、科技公司CEO |

⚠️ **配置文件中的 `keyFigures` 是参考列表，而非限制名单**。实际内容选取时，应：
- 优先关注配置中的重点人物
- 同时捕捉任何发表重要观点的相关领域领袖
- 以内容质量和影响力为最终标准

### 主题日配置参数

> 📁 **配置文件**: [`backend/config/day-config.json`](file:///Users/dq/Thoughts%20Rador/Thoughts-hunter/backend/config/day-config.json)

编辑此文件即可设定主题日，无需修改代码：

```json
{
  "dates": ["2026-01-20", "2026-01-21", "2026-01-22", "2026-01-23", "2026-01-24"],
  "event": "达沃斯世界经济论坛 2026",
  "rules": {
    "maxItems": 12,
    "freqBalance": false,
    "maxPerFreq": null
  },
  "focus": {
    "direction": "全球经济走向与AI治理",
    "keyFigures": ["Klaus Schwab", "Sam Altman"],
    "keyTopics": ["AI监管框架", "绿色转型"]
  }
}
```

### 主题日内容特征

| 内容类型 | 说明 | 预期占比 |
|---------|------|---------|
| **主题演讲** | 论坛主旨演讲、开幕致辞 | 20-30% |
| **圆桌对话** | 多人讨论、观点碰撞 | 20-30% |
| **专访** | 重点人物一对一访谈 | 15-25% |
| **宣言/声明** | 官方发布、共识声明 | 10-20% |
| **深度评论** | 分析师/学者即时评论 | 10-20% |

### 主题日标签

主题日发布的内容应添加特殊标签：

- `#DavosWeek` / `#达沃斯2026`
- `#ThemeDay` / `#主题日`
- 事件专属标签

## 🚨 内容去重规范 (Duplicate Prevention)

> ⚠️ **此规则对普通日和主题日均强制执行**

### 核心原则

- **零重复容忍**：已发布的内容**严禁再次发布**
- **来源唯一性**：同一原始来源（URL/YouTube ID）**仅允许发布一次**
- **唯一性校验前置**：所有内容写入前必须通过校验

### 去重检查清单

| 检查项 | 阻断级别 |
|-------|---------|
| 来源URL/视频ID 已存在 | **P0 强制阻断** |
| 标题相似度 ≥ 80% | **P0 强制阻断** |
| 同作者同频段7天内相似观点 | **P1 需审核** |
| 同日期同主题（普通日） | **P2 警告** |

## 🔍 来源质量标准（通用）

### ✅ 合格来源
- 权威榜单/奖项获得者（诺贝尔、普利策、图灵奖等）
- 权威媒体深度栏目（NYT、FT、The Atlantic、Foreign Affairs）
- 顶级头脑原始表达（TED、达沃斯、Lex Fridman播客等）
- 【主题日额外】活动官方频道、指定媒体合作伙伴

### ❌ 禁止使用
- 普通新闻报道（无深度分析）
- 社交媒体碎片化言论
- 未经验证的自媒体内容
- 商业公关/软文

---

## ⚠️ 真实性要求（不可妥协）

- **零容忍虚构**：严禁虚构来源、日期、引用或事实
- **可追溯性**：每条内容必须关联真实原始链接
- **双重核验**：自动化草稿需手动确认来源存在
